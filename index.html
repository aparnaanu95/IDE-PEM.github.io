Project Evaluation and Management

Course Code: 222ECE098

Home Syllabus Course Content

Course Overview

This course introduces the management aspects of project idea formulation, feasibility studies, costing, project appraisal and project financing.

Preamble

The objective of this course is to enable students to understand the management aspects involved in project idea formulation, feasibility studies, project report preparation, project costing, appraisal and funding.

Course Outcomes

  • Develop project ideas
  • Perform feasibility analysis of projects
  • Plan and estimate project costs
  • Carry out project appraisal
  • Identify appropriate funding sources
